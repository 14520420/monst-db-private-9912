# 🌟 おすすめ商品・ティア表サイト

URLを知っている人だけがアクセスできる、非公開寄りの商品紹介＆ティア表Webサイトです。

---

## ✅ 実装済み機能

### 🏆 ティア表
- S / A / B / C / D の5段階ティア表示
- 編集モードでドラッグ＆ドロップによるティア変更
- 未分類エリア
- 各カードをクリックして商品詳細を表示

### 📦 商品一覧
- カード型グリッドレイアウト
- カテゴリフィルター
- キーワード検索（商品名・説明・タグ対応）
- 商品詳細モーダル

### ✏️ 編集機能（編集モード）
- 商品の追加・編集・削除
- 画像URLプレビュー
- ティアランク設定
- タグ管理
- 価格・リンク・カテゴリ設定

### 🔒 非公開設定
- `<meta name="robots" content="noindex, nofollow">` — 検索エンジンへのインデックス拒否
- `robots.txt` — 全クローラーのアクセス拒否

## 🗄 データモデル

### `products` テーブル
| フィールド   | 型        | 説明                      |
|-------------|-----------|--------------------------|
| id          | text      | 一意ID                   |
| name        | text      | 商品名（必須）             |
| description | rich_text | 商品説明                  |
| image_url   | text      | 画像URL                  |
| price       | text      | 価格表示文字列             |
| link        | text      | 購入リンクURL              |
| category    | text      | カテゴリ名                 |
| tier        | text      | ティアランク（S/A/B/C/D/未分類）|
| tags        | array     | タグリスト                 |
| sort_order  | number    | 表示順                    |

### `tier_settings` テーブル
| フィールド   | 型      | 説明             |
|-------------|---------|-----------------|
| id          | text    | 一意ID           |
| tier_key    | text    | ティアキー        |
| label       | text    | 表示ラベル        |
| color       | text    | カラーコード      |
| sort_order  | number  | 表示順           |

---

## 🔗 API エンドポイント

| メソッド | URL                         | 説明            |
|---------|-----------------------------|----------------|
| GET     | `tables/products`           | 商品一覧取得    |
| POST    | `tables/products`           | 商品追加        |
| PUT     | `tables/products/{id}`      | 商品全更新      |
| PATCH   | `tables/products/{id}`      | ティア変更等    |
| DELETE  | `tables/products/{id}`      | 商品削除        |
